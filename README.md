# sitegeist/git-hooks

> Git-Hooks which are supporting our workflow and QA process.


| Prefix        | Description                                                      |
| ------------- | ---------------------------------------------------------------- |
| `pre-commit`  | Lints all `.js` files and aborts the commit if errors are found. |
